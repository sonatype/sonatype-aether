sonatype-aether by sonatype

Aether is a library for interacting with artifact repositories, powering dependency resolution in Apache Maven 3.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/sonatype/sonatype-aether

get the source code on GitHub : sonatype/sonatype-aether
